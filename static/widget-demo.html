🧠 Knowledge Brain Chat Widget

Add an AI-powered knowledge base chatbot to your website in 60 seconds

👋 Try it now! Click the chat button in the bottom-right corner

👇

🚀 Quick Start

Add this single line to your website's <head> or before </body>:

<script src="https://YOUR_VERCEL_URL.vercel.app/widget.js"></script>

That's it! The chat widget will automatically appear on your site.

🎨 Customization

You can customize the widget by adding these variables BEFORE the script tag:

<script> window.CHATBOT_API_URL = 'https://your-api.vercel.app'; window.CHATBOT_NAME = 'My Support Bot'; window.CHATBOT_AVATAR = '🤖'; window.CHATBOT_COLOR = '#ff6b6b'; window.CHATBOT_GREETING = 'Hey! How can I help you today?'; </script> <script src="https://YOUR_VERCEL_URL.vercel.app/widget.js"></script>

📋 Integration Steps

  1. Deploy your chatbot on Vercel (you already did this!)
  2. Copy the script tag from the Quick Start section above
  3. Paste it into your website's HTML before </body>
  4. Refresh your website — the chat bubble appears!
  5. Customize using the options above (optional)
